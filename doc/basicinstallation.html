Basic Installation

Once you've got hold of a CCNet binary distribution (see Obtaining), unzip it into a folder on your system. (If you're using Windows XP or 2003 with built in zip-file browsing, don't just use rely on this feature - actually unzip the distribution!)

Inside this folder you'll see sub-folders for the different applications that form CruiseControl.NET, as described in the CCNet Overview. There's also a documentation folder relevant to the version of CruiseControl.NET you obtained.

There's detailed help for each application linked from the documentation index.
