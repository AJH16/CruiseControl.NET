CruiseControl.NET build process

Steps:

  1. Initiate/Init/Setup/Start/PreBuild
  2. RunBuild/Build
  3. Finish/TearDown/PostBuild

PreBuild

  1. Load previous BuildResult from BuildHistory, if not previously loaded. a) if no previous build -> set to empty build result b) if last build is null -> load recent build c) if current build is not null -> last build = current build
  2. Create Build Result
  3. Label Build - generate new label from previous BuildResult

Build

  1. BuildResult.Start()
  2. Get Modifications from Source Control - from last modification date from previous BuildResult
  3. Execute Builder
  4. BuildResult.End()

PostBuild

  1. Send BuildEvent notification
  2. Save BuildResult to BuildHistory
