Installing and Configuring the Web Dashboard Application

The CCNet Web Dashboard Application is used for reporting summary details for a set of projects. To install it:

  1. Unzip the distribution as described in Basic Installation
  2. In IIS, map a virtual directory to the 'webdashboard' subfolder of your CCNet folder

The Dashboard needs to connect to each of Runners you want to report on, so make sure you set each Runner up for management (see Enabling Management in the Runner documentation).

To configure the dashboard, in the <projectURLs> section of the dashboard's 'web.config' file, add a <project> tag for each runner you want to monitor. The value of the tag is the management URL for the runner. Here is an example <projectURLs> section:

<projectURLs>
	<project url="tcp://localhost:1234/CruiseManager.rem"/>
	<project url="tcp://myotherbuildserver:2345/CruiseManager.rem"/>
</projectURLs>
