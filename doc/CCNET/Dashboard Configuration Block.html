CruiseControl.NET : Dashboard Configuration Block
This page last changed on Mar 07, 2005 by mroberts.

The root tag in the Dashboard Configuration file is the <dashboard> tag.

The <dashboard> tag contains a <remoteServices> section and a <plugins> section.

Example:

<dashboard>
    <remoteServices>
    .
    .
    </remoteServices>

    <plugins>
    .
    .
    </plugins>
</dashboard>
