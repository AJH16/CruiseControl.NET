CruiseControl.NET : Trigger Blocks
This page last changed on Oct 26, 2004 by mroberts.

Trigger blocks allow you to specify when CruiseControl.NET will start a new integration cycle.

Typically, you'll want to use a Polling Interval Trigger. Also useful is the Force Build Schedule Trigger for implementing daily builds.

Types of Integration Trigger Block are:

  • Force Build Interval Trigger
  • Force Build Schedule Trigger
  • Polling Interval Trigger
  • Polling Schedule Trigger
