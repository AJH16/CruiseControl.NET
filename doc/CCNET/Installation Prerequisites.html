CruiseControl.NET Server

To get CruiseControl.NET server up and running, you need to have the following environments set up and installed:
  • Microsoft.NET Framework Version 1.0 or 1.1 (download)
  • Mono (download)

CruiseControl.NET Web Application / Dashboard

  • Microsoft.NET Framework Version 1.0 or 1.1 (download)
  • ASP.NET-enabled web server / IIS

Developer desktop running CCTray

  • Microsoft.NET Framework Version 1.0 or 1.1 (download)

Supported Tools

CruiseControl.NET is designed to integrate with the following source control systems and build tools.
  • CVS (http://www.cvshome.org/)
  • Microsoft Visual Source Safe (http://msdn.microsoft.com/ssafe/)
  • SubVersion (http://subversion.tigris.org/)
  • StarTeam (http://info.borland.com/techpubs/starteam/)
  • PVCS (http://www.merant.com/Products/ECM/vm/home.asp)
  • NAnt (http://nant.sourceforge.net)
  • NUnit (http://nunit.org)
  • FxCop (http://gotdotnet.com/team/fxcop/)
