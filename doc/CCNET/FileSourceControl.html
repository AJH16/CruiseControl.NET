'File' Source Control Configuration Example

You can use the 'File' Source Control plugin to check for modifications on a directory accessable by the build server. You can use either directories on 'mapped' drives (local or remote), or UNC paths (remote). 
<sourcecontrol type="filesystem">
	<repositoryRoot>c:\mycode</repositoryRoot>
</sourcecontrol>
