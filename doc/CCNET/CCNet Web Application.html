CruiseControl.NET : CCNet Web Application
This page last changed on Apr 21, 2004 by orogers.

The CCNet Web Application is used for reporting the status of all the builds for a project. To install it:

  1. Unzip the distribution as described in Basic Installation
  2. In IIS, map a virtual directory to the 'web' subfolder of your CCNet folder

To configure the webapp, set the 'logDir' value in the 'appSettings' section of the web.config file. By default, its set to be the 'log' sub-directory of the web folder, but if your Server's 'xmllogger' task is creating its log files elsewhere you'll need to update this.
