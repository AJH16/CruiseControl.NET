CruiseControl.NET : Key Features
This page last changed on Oct 13, 2004 by mroberts.

CruiseControl.NET brings the benefits of automated Continuous Integration to .NET projects.

  • Integration with NAnt and NUnit.
  • Integration with major open source and proprietary Source Control systems:
    • CVS Source Control Block
    • Subversion Source Control Block
    • Visual Source Safe Source Control Block
    • Perforce Source Control Block
    • StarTeam Source Control Block
    • PVCS Source Control Block
    • Rational ClearCase Source Control Block
    • SourceGear Vault Source Control Block
  • Multiple visibility options:
    • ASP.NET enabled Web Applications
    • HTML and plain-text email
    • System Tray notification
  • Customization options through pluggable architecture and source code availability.
