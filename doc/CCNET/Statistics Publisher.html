CruiseControl.NET : Statistics Publisher
This page last changed on Apr 11, 2006 by akumar.

The publisher can be used to collect and update statistics for each build in a file. Some of the statistics which would be collected are build durations and test count.

At present the configuration would be very minimal, just an empty <statistics /> element in the publishers section.

<publishers>
     <statistics />
</publishers>

This would generate a statistics.csv and statistics.xml file in the artifact directory.

Expect the document to be updated on how to get some charts for these data on the project report page soon!
