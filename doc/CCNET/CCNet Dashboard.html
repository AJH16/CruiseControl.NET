CruiseControl.NET : CCNet Dashboard
This page last changed on Apr 21, 2004 by orogers.

The CCNet Web Dashboard Application is used for reporting summary details for a set of projects. To install it:

  1. Unzip the distribution as described in Basic Installation
  2. In IIS, map a virtual directory to the 'webdashboard' subfolder of your CCNet folder

The Dashboard needs to connect to each of Servers you want to report on, so make sure you set each Server up for management (see Enabling Management in the Server documentation).

To configure the dashboard, in the <projectURLs> section of the dashboard's 'web.config' file, add a <project> tag for each Server you want to monitor. The value of the tag is the management URL for the Server. Here is an example <projectURLs> section: 
<projectURLs>
	<project url="tcp://localhost:1234/CruiseManager.rem"/>
	<project url="tcp://myotherbuildserver:2345/CruiseManager.rem"/>
</projectURLs>
