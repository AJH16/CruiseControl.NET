CruiseControl.NET : Force Build Interval Trigger
This page last changed on Oct 26, 2004 by mroberts.

The Force Build Interval Trigger is used to specify an interval, or period of time, between integration cycles. A build will always be started even if no changes have occurred to source control.

Examples

Minimalist example: 

<forceBuildInterval />

Full example: 
<forceBuildInterval seconds="30" />

Configuration Elements:

Node Description Type Required Default
seconds The number of seconds after an integration cycle completes before triggering the next integration cycle. int false 60
