CruiseControl.NET : Force Build Schedule Trigger
This page last changed on Oct 26, 2004 by mroberts.

The Force Build Schedule Trigger is used to specify that an integration should be run at a certain time each day. A build will always be started even if no changes have occurred to source control.

Example

<forceBuildSchedule time="23:30" />

Configuration Elements:

Node Description Type Required Default
time The time of day that the build should run at. The time should be specified in a locale-specific format (ie. H:mm am/pm is acceptable for US locales.) string true n/a
