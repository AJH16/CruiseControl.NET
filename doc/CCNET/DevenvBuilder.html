CruiseControl.NET : DevenvBuilder
This page last changed on May 05, 2004 by orogers.

Most complex build processes use NAnt or MSBuild to script out the build process. However, for simple projects that just need to compile a Visual Studio.NET solution, the DevenvBuilder provides an easy way to get the source integrated with CruiseControl.NET.

Sample Configuration

<build type="devenv" solutionfile="D:\dev\MyProject\MyProject.sln" configuration="debug" />

Configuration Elements:

Node Description Type Required
solutionfile Path of the solution file to build string true
configuration Solution configuration to use with the build (Not case sensitive) string true
executable Path to "devenv.com". By default, CC.NET will try to load this path from the registry. If both VS.NET 2003 and VS.NET 2002 are installed, CC.NET will try using VS.NET 2003. In this case, if you need to use VS.NET 2002, you should specify this property to point to the location of "devenv.com" for VS.NET 2002. string false
buildTimeoutSeconds Number of seconds to wait before assuming that the process has hung and should be killed. The default value is 600 (10 minutes) int false

Note: this builder does require you to have Visual Studio .NET installed on your integration server.
