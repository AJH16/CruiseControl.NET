Release Distributions

CruiseControl.NET release distributions are available from Sourceforge. If your company does not let you access Sourceforge, please email us

Continuous Integration Distributions

Distributions are automatically created by the CruiseControl.NET server on CCNetLive every time that new code is checked into the CruiseControl.NET CVS repository. You can download the latest CruiseControl.NET distributions from here.

Installation

Once you have downloaded the distribution, follow the installation instructions to start installing CruiseControl.NET.
