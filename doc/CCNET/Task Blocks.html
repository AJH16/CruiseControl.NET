CruiseControl.NET : Task Blocks
This page last changed on May 19, 2005 by orogers.

Types of Task Block are:

  • Build Publisher
  • Email Publisher
  • Executable Task
  • File Merge Task
  • ForceBuildPublisher
  • NAnt Task
  • NUnit Task
  • Visual Studio Task
  • Xml Log Publisher

Publishers are now interchangeable with Tasks. These tasks can be included either in the <tasks> section or the <publishers> section of the ccnet.config file depending on whether they should be run in or after the build.
