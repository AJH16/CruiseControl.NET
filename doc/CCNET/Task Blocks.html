CruiseControl.NET : Task Blocks
This page last changed on Mar 16, 2005 by mroberts.

Types of Task Block are:

  • Executable Task
  • File Merge Task
  • NAnt Task
  • NUnit Task
  • Visual Studio Task

Publishers can also be used interchangeably as Tasks. Include any Publisher in the Task Block to have it run the specified order.

  • Build Publisher
  • Email Publisher
  • Executable Publisher
  • ForceBuildPublisher
  • Net Send Publisher
  • Xml Log Publisher
