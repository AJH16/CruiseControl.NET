CruiseControl.NET Documentation

Welcome to the CruiseControl.NET (CCNet) Documentation!

  • If you are new to Continuous Integration (CI), start with Introduction to Continuous Integration.
  • If you are experienced with automated CI, you can skip straight to the CruiseControl.NET Overview.
  • The rest of the documentation is split into 2 sections - one for users : CruiseControl.NET Detailed Help ; and the rest for CCNet developers : CruiseControl.NET Developer documentation.

This documentation is a work in progress until we reach 1.0 status - if an item is mentioned but doesn't have an actual page, then it is due to be completed in a later release.
