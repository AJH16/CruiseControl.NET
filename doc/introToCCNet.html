CruiseControl.NET Overview

So you know about automated Continuous Integration, and want to know about what CruiseControl.NET does about it? Well this page is the right place to come.

CruiseControl.NET (CCNet) contains several applications. The core one is the CI 'runner' which provides the automation of your master build, as described in the original CruiseControl article. The Runner offers several key features:

  • Integration with the NAnt build system
  • Integration with several Source Control systems
  • Remote management and visibility

The remaining applications offer reporting and management. They are

  • CCNet Web App - The main CruiseControl.NET reporting application (an ASP.NET web app), which gives a breakdown for each of the builds for a project.
  • Dashboard - An ASP.NET monitoring application that gives you status information for your project, or set of projects on your network.
  • CCTray - A client System Tray application that enables you to see the state of your build from any machine on your network.
