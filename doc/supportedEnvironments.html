Supported Environments

CruiseControl.NET supports the following environments:

Runner Environment

  • Any Windows-based .NET 1.0 or 1.1 environment, using the Microsoft standard .NET 1.0 and 1.1 runtimes

Web Environment

  • Any ASP.NET / Microsoft IIS environment, under the same .NET conditions as the runner environment

Source Control Environment

  • CVS
  • Microsoft Visual SourceSafe
  • Perforce
  • PVCS
  • StarTeam

Build Environment

  • NAnt

We welcome feedback and suggestions about other environments - please contact us via the User mailing lists, as described on the CruiseControl.NET Website here
